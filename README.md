# 🛒🛒 TiendaVirtualMovil -- Android Studio (Panda)

### Este es mi proyecto de desarrrollo de software para la evidencia del **SENA**
### Esta aplicación móvil es el Frontend que se conectara con una API y una base de datos en MySQL. 
## 👩‍💻 Tecnologías Utilizadas
* **IDE:** Android Studio Panda
* **Lenguaje:** Kotlin 1.9+
* **Gestor de Dependencias:** Gradle con Version Catalog
* **Comunicación de Red:** Retrofit 2.9.0
* **Serialización de datos:** Google Gson

## 🗂️ Arquitectura del proyecto
Utilice una Arquitectura en capas enfocada en la mantenibilidad:
* **`data`**: Que contiene los repository (Producto, Usuario, Pedido, pago) encargados de la lógica de datos
* **`model`**: Que define las entidades del sistema (clases de datos)
